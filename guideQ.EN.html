Iqra is one of the hundreds of software applications made for reading our Noble Book, the Qur'an. The main features that distinguish it from other software are: marking the root of a word on the Qur'an text, showing all pages where a word appears in the Qur'an on a single screen, and working in an offline environment.

For ease of memorization and recitation, we used standard Mushaf pages.

When the Mushaf is opened, if the screen is wide enough both the Qur'an page and the translation appear. If both cannot fit together, only one is shown (See: Panels) While the Mushaf is open, it is possible to go to the next or previous page, or specify another surah, juz, verse, or page number.


All search operations are managed by entering information into the single box in this dialog that resembles the start page. Based on the entered information, the software decides which type of search or navigation is required. In the same dialog, it is also possible to search spoken words using the microphone. Voice search is applicable only for Arabic text.

The meaning of the buttons visible at the top right:

The root and meaning of Arabic words can be found by clicking with a mouse on desktop or tapping on mobile. Verses and words found in the search panel (Mucem or Guide) are highlighted in color in that panel. On narrow screens both verse and translation columns do not fit together; the T menu (or screen scrolling) is needed to switch between verses and translations. Fonts scale with screen width, so holding the phone sideways provides a better view.


In this image, the context menu is shown after two words are selected. For the context menu: if there is a mouse, right-click; otherwise long-press the word or touch with two fingers. The selected text is copied to the clipboard or sent to search modules.

2. Search methods

The software supports three types of search:

  1. Plain text search (in three languages) – Guide
  2. Search by Arabic word root – Mucem
  3. Search for verses similar on a word-by-word basis

2.1 Guide (Three languages)

The plain text search in Arabic text or in the Turkish-English translation was presented as a graduation project by Abdurrahman Rajab (Completion date: June 2020)

2.2 Mucem (Arabic only)

The Mucem method works based on the roots of selected Arabic words. This method is designed for those who know or want to learn the Qur'anic language. Each of the 604 boxes shown on the page corresponds to a page from the Qur'an. A searched root does not appear at all on white pages. The color of each box is proportional to the number of words on the corresponding page. Details are on the related page

3. Software capabilities

Because it is a PWA (Progressive Web Application), its features include:

  • When opened from a web page its components are automatically downloaded
  • No manual updates required; new versions are downloaded automatically
  • Can open in its own window like an app
  • Can operate offline without an Internet connection
  • Compatible with every screen size from desktop to phone

4. Frequently asked questions

  • What need gave rise to this project?

    In Qur'anic recitation, one encounters verses that resemble each other. To explain one verse by another, it is necessary to quickly move between similar verses and see them all together.

  • What distinguishes Iqra from other Qur'an software?

    * Showing the roots of Arabic words
    * Working independently from the internet
    * Automatically downloading new versions

  • What does Mucem mean?

    An index showing in which verses each word occurs

  • Source of Mucem data?

    corpus.quran.com (GNU License is included below)

  • What does the Mucem table mean?

    Each page is a box, each juz is a row

  • Why the Medina mushaf?

    We chose the Medina mushaf for three reasons:
    * Tajwid marks make reading easier
    * Most common international writing style
    * Compatible with different qira'at narrations

5. Use cases

  • Daily reading: I bookmark where I left off and continue the next day from the bookmarked page
  • Research: meanings and roots of words, Mucem, similar verses
  • Finding verses: by typing on the keyboard, or by voice search
  • Topics: viewing all verses of a selected group together
  • Lessons: direct access to verses from pages written on various topics
  • External resources: Missing features such as additional translations, audio recitation, qira'at differences, manuscripts are completed by links to other sites. If the relevant site has a mushaf page display feature, it appears in the page menu.
