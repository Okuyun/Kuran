27:88 sun`allâhillezî etqane kulle şey
İşte bu, her şeyi sağlam yapan Allah'ın sanatıdır.

Yüce Allah, kendi sanatı olan yaratılışı anlamamız için bize muhtelif sanatlar öğretiyor. Mimarlıktan müziğe, hattatlıktan mutfağa, terzilikten yazılıma... her yaptığımız iş bir sanat olarak değerlendirilmeli, öyle okunmalı.

    Bu yazılımın evrimi üç safhada gelişti:
  • Easy View   Mac – 1991  Java – 2004
  • Iqra 3   Java – 2011  Web – 2019
  • Iqra 4   Web – 2020

Easy View

Iqra projesi 1991'de (henüz web yok iken) basit bir Mac yazılımı olarak başladı. Birbiriyle ilişkili çok sayıda metin dosyasının iç yapısını kullanarak bilgiye hızlı erişim sağlıyordu. Kuran metni ve meali, içine eklenen yapısal elemanlarla, bu yazılıma uyarlandı. Sonuç olarak, Kuran'ın istenen ayetine bir anda erişebiliyor ve aranan kelimeyi birkaç saniye içinde bulabiliyorduk.

1. Mac sürümü – 1991

Mac daha ilk günden yerelleştirmeye uygun tasarlanmıştı. Her pencerede farklı bir dil kullanabiliyorduk. Böylece Kuran metnini ve mealini aynı ekranda iki pencerede izleme imkanı doğdu. Basit bir DOS programı olan Salsabil'den Arapça metin, Diyanet mealinden Türkçe metin kullandık. Think Pascal ile geliştirilen bu yazılım sadece Mac OS altında çalışıyordu.

Yazılımın Arapça (ve diğer diller) ile uyumunu gösteren bir ekran resmi bulamadım. Ama bu özelliği sayesinde ödül kazandığını burada eklemeliyim.



2. Windows sürümü – 1994

Bilkent'te öğretim üyesi olan David Davenport yazılımı Windows'a uyguladı ama uygulama çok başarılı olmadı. Mac OS'de çalışan bir çok özelliği Windows'a taşıyamadık. Arapça metin şöyle dursun, Türkçeye özgü Ğ ve Ş bile doğru görünmüyordu, çünkü işletim sisteminde bu özellikler yoktu. Yine de, PC dünyasında İngilizce meale en hızlı ulaşan yazılım oldu.



3. Java sürümü – 2004

Yazılımın Java'ya taşınması ile, Easy View bütün masa-üstü ve diz-üstü sistemlerde çalışabilir oldu. Dil sorunu büyük ölçüde çözüldü.

Java ayarları doğru yapılınca bu penceredeki Türkçe harfler elbette doğru görünüyordu. (Şu sıralar hiç Java kullanmadığım için ayarları yapmadım) Sonraki sene Unicode sayesinde dil sorunu kökten çözüldü.


Iqra 3

Quranic Arabic Corpus çıkınca, Arapça metne hakim olmaya başladık. Genel amaçlı metin işleme yazılımı Easy View, sadece Kuran amaçlı Iqra yazılımına dönüştü. Bu arada sayfa düzenini standart mushaflara uygun hale getirdik. Kelimelerin kökleri verildiği için artık Mucem yazılımı da gerçekleşebilirdi.

V3.00 – 2011 (ilk Java sürümü)
V3.05 – 2017 (son Java sürümü)

Yukarıdaki pencereyi görmek için cihazınızda Java kurulu olmalı ve şu iki dosyayı indirmelisiniz: jar file ve me_quran font

Lakin Java ile yazılım geliştirmek zordu, 7 yılda fazla ileri gidemedik. Mucem bir türlü hayalden hakikate çıkamadı. Üstelik o yıllar mobil cihazların tırmanış zamanıydı ve standart Java bu cihazlarla uyumlu değildi. Mobil cihazlarda çalışması için web sayfalarına geçmeliydik.
