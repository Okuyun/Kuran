Easy View

Iqra projesi 1991'de (henüz web yok iken) basit bir Mac yazılımı olarak başladı. Metin dosyalarının iç yapısını kullanarak hızlı erişim sağladığı için, Kuran metnine ve mealine aynı hızda erişebiliyorduk.

Mac sürümü – 1991

Mac daha ilk günden yerelleştirmeye uygun tasarlanmıştı. Her pencerede farklı bir dil kullanabiliyorduk. Böylece Kuran metnini ve mealini aynı ekranda iki pencerede izleme imkanı doğdu. Basit bir DOS programı olan Salsabil'den Arapça metin, Diyanet mealinden Türkçe metin kullandık. Think Pascal ile geliştirilen bu yazılım sadece Mac OS altında çalışıyordu.

Yazılımın Arapça (ve diğer diller) ile uyumunu gösteren bir ekran resmi bulamadım. Ama bu özelliği sayesinde ödül kazandığını burada eklemeliyim.



Windows sürümü – 1994

Yazılımın Windows'a uygulanması çok başarılı olmadı. Mac OS'de çalışan bir çok özelliği Windows'a taşıyamadık. Arapça metin şöyle dursun, Türkçeye özgü Ğ ve Ş bile doğru görünmüyordu.



Java sürümü – 2004

Yazılımın Java'ya taşınması dil sorununu bütün masa-üstü ve diz-üstü sistemlerde çözdü.

Java ayarları doğru yapılınca bu penceredeki Türkçe harfler elbette doğru görünüyordu. Bunca zaman sonra bu kadar çalışması bile tebrike şayan! Sonraki sene Unicode sayesinde dil sorununu kökten çözüldü.


Iqra 3

Quranic Arabic Corpus çıkınca, Arapça metne hakim olmaya başladık. Kelimelerin kökleri de verildiği için artık Mucem yazılımı gerçekleşebilirdi.

V3.0 – 2011 (ilk Java sürümü)
V3.05 – 2017 (son Java sürümü)

Yukarıdaki pencereyi görmek için cihazınızda Java kurulu olmalı ve şu iki dosyayı indirmelisiniz:
jar file ve me_quran font

Lakin Java ile yazılım geliştirmek zordu, 7 yılda fazla ileri gidemedik. Mucem bir türlü hayalden hakikate çıkamadı. Üstelik o yıllar mobil cihazların tırmanış zamanıydı ve standart Java bu cihazlarla uyumlu değildi. Mobil cihazlarda çalışması için web sayfalarına geçmeliydik.


Mobil sürüm – 2019 (JavaScript)

V3.06 Web sayfası (ilk mobil sürüm)

Bugün çok ilkel görünen bu sayfada istenen bütün data mevcut. İlk defa kendi web sayfamızda Kuran metni ve meali bir arada göründüğü için bu sayfa çok önemli bir adım: Artık dosya indirmek ve yazılım kurmak tarihe karıştı. Öte yandan, bu ilk sürümde hatalar vardı. Arapça metne bakınca, besmele ile ilk ayetin karıştığını görebilirsiniz.
